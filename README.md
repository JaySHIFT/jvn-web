# JVN Portfolio

A simple personal portfolio website built with plain HTML and CSS, showcasing a web developer's skills, story, and contact details.

## Overview

This project is a static portfolio for `James Web` (JVN), featuring a clean responsive layout, strong visual hierarchy, and polished presentation. It includes a home page, about page, contact page, and supporting assets.

## Pages

- `index.html` — Home page with hero section, feature highlights, and quick contact call-to-action.
- `about.html` — About page explaining the developer's approach, growth mindset, and front-end skill set.
- `contact.html` — Contact page with an email form and additional contact links.

## Assets

- `style.css` — Main stylesheet for layout, typography, colors, and responsive design.
- `img/` — Image assets including the site favicon and logo.

## Features

- Responsive layout designed for desktop and mobile
- Semantic HTML structure for readability and accessibility
- Custom fonts loaded from Google Fonts
- Contact form with mailto action
- Navigation between pages and sections

## Usage

1. Open `index.html` in a web browser.
2. Use the navigation links to browse the portfolio.
3. Submit the contact form or use the provided email and social links to connect.

## Notes

- The contact form uses `mailto:` and will open the visitor's default email application.
- Replace the placeholder email address in `contact.html` if needed.
